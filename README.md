# Sync Service for Yandex Disk

Сервис автоматической синхронизации локальной папки с Яндекс Диском.

## Функционал

* Автоматическая загрузка новых файлов из локальной папки в облако.
* Обновление изменённых файлов (по временной метке).
* Опциональное удаление файлов из облака (настраивается через конфигурацию).
* Логирование всех операций.
* Защита от случайного удаления файлов (настраиваемая).

## Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/romanima/sync-service-yandex-disk.git
   cd sync-service-yandex-disk
Создайте виртуальное окружение:

bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или .venv\Scripts\activate  # Windows
Установите зависимости:

bash
pip install -r requirements.txt
Настройте окружение:

bash
cp .env.example .env
Отредактируйте файл .env под свои нужды (см. раздел «Настройка»).

Использование
Запустите сервис:

bash
python main.py
Сервис будет работать в бесконечном цикле, синхронизируя файлы с заданным интервалом.
